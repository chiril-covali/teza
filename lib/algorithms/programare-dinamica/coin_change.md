# Problema Restului (Coin Change)

Slug: programare-dinamica_coin_change
Categorie: Programare Dinamică

## Descriere
Given a set of categories of coins C and an amount of money S, the goal is: to give change for S but to use a minimum number of coins. Suppose each category of coin has an infinite number of pieces.

- Folosește cel puțin o buclă for în implementare.
