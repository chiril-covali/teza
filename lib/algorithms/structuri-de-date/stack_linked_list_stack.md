# Stivă Înlănțuită Listă Stivă

Slug: structuri-de-date_stack_linked_list_stack
Categorie: Structuri de Date

## Descriere
This is an implementation of a stack, based on a linked list. A stack is a linear data structure that works with the LIFO (Last-In-First-Out) principle.
