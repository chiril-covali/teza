# Verificare Putere a lui 2

Slug: manipulare-biti_is_power_of_2
Categorie: Manipulare Biți

## Descriere
This code will check whether the given number is a power of two or not. A number will be a power of two if only one bit is set and rest are unset.

## Observații din implementare
- Implementarea folosește funcții arrow/funcții compacte.
