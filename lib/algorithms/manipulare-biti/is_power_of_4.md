# Verificare Putere a lui 4

Slug: manipulare-biti_is_power_of_4
Categorie: Manipulare Biți

## Descriere
Checks whether the given number is a power of four or not. A number is considered a power of four if and only if there is a single '1' bit in its binary representation,

## Observații din implementare
- Implementarea folosește funcții arrow/funcții compacte.
