# Floyd Warshall

Slug: grafuri_floyd_warshall
Categorie: Grafuri

## Descriere
Compute the shortest path for all pairs of nodes for a graph without negative weight edges. The input graph is a adjacency matrix, where graph[i][j] holds the weight of edges a->b. If the edge does not exist, the value in the matrix is Infinity.

## Observații din implementare
- Folosește cel puțin o buclă for în implementare.
- Implementarea folosește funcții arrow/funcții compacte.
