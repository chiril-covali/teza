# Prim

Slug: grafuri_prim
Categorie: Grafuri

## Descriere
Compute a minimum spanning tree(MST) of a fully connected weighted undirected graph. The input graph is in adjacency list form. It is a multidimensional array of edges. graph[i] holds the edges for the i'th node. Each edge is a 2-tuple where the 0'th item is the destination node, and the 1'th item is the edge weight.

## Observații din implementare
- Folosește cel puțin o buclă while în implementare.
- Folosește cel puțin o buclă for în implementare.
