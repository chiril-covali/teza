# Johnson

Slug: grafuri_johnson
Categorie: Grafuri

## Descriere
Compute the shortest path for all pairs of nodes. The input graph is in adjacency list form. It is a multidimensional array of edges. graph[i] holds the edges for the i'th node. Each edge is a 2-tuple where the 0'th item is the destination node, and the 1'th item is the edge weight. Returned undefined if the graph has negative weighted cycles.

## Observații din implementare
- Folosește cel puțin o buclă for în implementare.
- Implementarea folosește funcții arrow/funcții compacte.
