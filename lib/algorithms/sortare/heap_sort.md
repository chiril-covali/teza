# Heap Sortare

Slug: sortare_heap_sort
Categorie: Sortare

## Descriere
is a comparison-based sorting algorithm that uses a binary heap data structure to repeatedly select and remove the maximum (for max-heap) or minimum (for min-heap) element and place it at the end of the sorted array.

## Observații din implementare
- Folosește cel puțin o buclă for în implementare.
- Implementarea folosește funcții arrow/funcții compacte.
