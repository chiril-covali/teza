# 🚀 **Număr Pronic (Pronic Number)**

## 📝 **Descriere**

Un **Număr Pronic** (sau număr oblong) este un număr care reprezintă produsul a două numere întregi consecutive, având forma $n(n + 1)$. Geometric, aceste numere reprezintă aria unui dreptunghi în care lungimea este cu o unitate mai mare decât lățimea. Ele sunt strâns legate de numerele triunghiulare (un număr pronic fiind dublul unui număr triunghiular).

## 🖼️ **Reprezentare Vizuală**

![Pronic Number](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c2/Pronic_number_12.svg/200px-Pronic_number_12.svg.png)

```text
Exemplu: 12 (3 x 4)
● ● ● ●
● ● ● ●
● ● ● ●

Șirul numerelor pronice:
0 * 1 = 0
1 * 2 = 2
2 * 3 = 6
3 * 4 = 12
4 * 5 = 20
```

## ⚖️ **Avantaje și Dezavantaje**

| Avantaj | Dezavantaj |
| :--- | :--- |
| ✅ **Generare Rapidă:** Calculul este instantaneu folosind o formulă polinomială simplă. | ⚠️ **Raritate:** Doar o mică fracțiune din numerele întregi sunt pronice. |
| ✅ **Relații Matematice:** Oferă punți de legătură între numerele pătratice și cele triunghiulare. | ❌ **Aplicabilitate Limitată:** Utilizat mai mult în teoria numerelor decât în ingineria software aplicată. |

## 🔢 **Analiză Matematică și Complexitate**

Proprietatea de bază: $P_n = n^2 + n$.

| Tip Complexitate | Valoare |
| :--- | :--- |
| **Timp (Time)** | $O(1)$ |
| **Spațiu (Space)** | $O(1)$ |
